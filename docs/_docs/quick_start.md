### Prerequisites

- [Python 3](https://python.org)
- [Git](https://git-scm.com)

### Installing

Run `git clone https://github.com/C1200/CorLang.git`

### Running

To run the CorLang CLI run:
```bash
cd CorLang
python3 lib/cli.py [file]
```

The CLI can optionally take in a file to execute.
